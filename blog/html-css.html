inline versus inline-blocks

In Sprint 2, we were asked to choose and research a topic from a list of technical topics.

I decided to look into the difference between inline and inline-blocks, as it wasn't something I had picked up during my other readings.

Note that I have drawn on the work of Samantha Ming and have borrowed some pictures from her website.

inline

An inline element does not start on a new line and it will only take up as much width as its content. If you try to set the width or height, or margin or padding, this wonâ€™t have any effect.

Some examples of elements that have a default inline property are:

  • span
  • a
  • img

Most tags that are used for formatting as also inline:

  • em
  • strong
  • small

inline-block

These are similar to inline elements, in that they don't start on a new line. However, unlike inline elements, you can set height and width values.

Some examples of elements that have a default inline property are:

  • span
  • a
  • img

Most tags that are used for formatting as also inline:

  • em
  • strong
  • small
